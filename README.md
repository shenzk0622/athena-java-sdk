### Maven dependency

Maven dependency configuration:

```xml

<dependency>
    <groupId>com.uniubi.cloud</groupId>
    <artifactId>athena-java-development-kit</artifactId>
    <version>${latestTagName}</version>
</dependency>
```

Use the latest release tag name as the version.
